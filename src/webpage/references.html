References for Embedded System Libraries

This document contains references to the libraries used in the embedded system code.

IOT:

  • Wire (I2C communication)
  • MAX30105 (Heart Rate and Blood Oxygen Sensor)
  • spo2_algorithm (SpO2 Calculation Algorithm)
  • Particle (IoT Device Library)

Code References

  • Node.js - JavaScript runtime for building the server.
  • HTTP Server - A simple, zero-configuration command-line HTTP server.
  • Express.js - Web application framework for Node.js.
  • MongoDB - NoSQL database used to store user data and readings.
  • JSON Web Tokens - For securely transmitting information between parties as a JSON object.
  • Chart.js - Simple yet flexible JavaScript charting for designers & developers.

Technology References

  • Particle Photon and Argon Boards - IoT devices used in the project.
  • MAX30102 Pulse Detection Blood Oxygen Sensor - Sensor used for tracking heart rate and blood oxygen levels.

General Resources

  • W3Schools - Resource for learning HTML, CSS, and JavaScript.
  • MDN Web Docs - Detailed documentation on HTML, CSS, JavaScript, and Web APIs.
